# Regional Sea Surface Height Exploration and Analysis
Explore some NASA datasets to understand trends in sea surface height
over the past decades.

## Downloading datasets
The following datasets are used in the project, and are publically
available from NASA:
* [NASA_SSH_REF_SIMPLE_GRID_V1](https://podaac.jpl.nasa.gov/dataset/NASA_SSH_REF_SIMPLE_GRID_V1#)
* [MERGED_TP_J1_OSTM_OST_GMSL_ASCII_V52](https://podaac.jpl.nasa.gov/dataset/MERGED_TP_J1_OSTM_OST_GMSL_ASCII_V52#)

## Usage
Once you have downloaded the datasets, you can begin using the scripts.

1. Install dependencies
		pip install -r requirements.txt
2. Run any script
		python3 simple_sealevel.py
		python3 gridded_analysis.py
